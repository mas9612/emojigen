# emojigen

Generate Slack emoji.

```
$ ./emojigen -h                                                                                                                                                                      (master)
Usage of ./emojigen:
  -color string
        Font color code omitted precede # character. Default: EC71A1FF
  -font string
        Font family. Default: notosans-mono-bold. Valid family is one of [notosans-mono-bold, mplus-1p-black, rounded-x-mplus-1p-black, ipamjm, LinLibertine_RBah, aoyagireisyoshimo].
  -out string
        Output filename. Required.
  -public
        Whether new emoji will be make public. Default: false.
  -text string
        Emoji text. Required.
```
